iOS向Unity传值

iOS向Unity传值主要通过void UnitySendMessage(const char* obj, const char* method, const char* msg);这个方法
这个方法可以传入三个参数:

  • 参数一为unity脚本挂载的gameobject
  • 参数二为unity脚本中要调用的方法名
  • 参数三为传递的数据，*注意：传递的数据只能是char 类型

总结

总之就是删除Unity默认的启动时机

这样 iOS部分的开发 和 Unity部分的开发就可以同时进行了
